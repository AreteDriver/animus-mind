# Animus World Model Architecture

**Version:** 1.0
**Status:** Proposed

## Executive Summary

Animus evolves from an assistant into a persistent world-modeling intelligence.

- Store events
- Build world state
- Detect patterns
- Forecast outcomes
- Support decisions
